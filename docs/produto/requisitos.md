# Requisitos

## Requisitos Funcionais

1. Gerar Dashboard principal que contenham:
   - Quantidade total de alunos
   - Quantidade de aulas cadastradas
   - Taxa média de presença
   - Alertas de baixa frequência

2. Gerenciamento de turmas:
   - Criar turmas
   - Editar turmas 
   - Remover turmas
   - Listar alunos por turma
   - Descrever cada turma
     -  nome, código, horário e quantidade de alunos

3. Controle de frequência:
   - Registrar presença e falta
   - Registrar data e horário da aula
   - Gerar histórico de frequência 
     - Por aluno e por turma
   - Validaçar a frequência

4. Relatórios:
   - Registrar frequência por aluno
   - Calcular o percentual de presença
   - Gerar lista de faltosos
   - Gerar histórico de presença por turma

5. Solicitar autenticação para acesso aos dados sensíveis.

## Requisitos Não Funcionais

- Deploy usando Docker desde o início
