## Diretrizes visuais
- Layout com foco em clareza e hierarquia de informação.
- Navegação lateral ou superior para Dashboard, Turmas, Frequência e Relatórios.
- Cards de resumo para indicadores principais.
- Feedback visual claro para ações de salvar, excluir e registrar presença.

## Páginas principais

- **Dashboard**: métrica rápida, alertas e gráficos de presença.
- **Turmas**: lista de turmas com busca e filtro.
- **Frequência**: registro de presença por aluno, data e aula.
- **Relatórios**: exportação de dados e visualização de faltosos.
